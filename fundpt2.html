  • What is the difference between single, double, and backtick quotes for strings?

    • Togge Answer

    Backticks are used for Template Literals.

  • How do you embed variables/expressions into a string?

    • Togge Answer

    Enclose the string in backticks, and place the variable/expression inside ${}

  • How do you escape characters in a string?

    • Togge Answer

  • What is the difference between slice/substring/substr?

    • Togge Answer

    substr can give inconsistent results. Modern browers allow using a negative start index to indicate the number of characters from the end of the string, but IE8 and lower treat a negative start index as 0. substring's parameters are reversable, as it will always use its smallest parameter value as the start index and largest value as the stop index. substring will treat a negative start index as 0. slice extracts from the end of the string if the starting index is negative.
