Javascript Study Guide
Fundamentals: Part 1
  • What are three different ways to declare a variable?

    • Togge Answer

    let, const, and var keywords

  • When should each be used?

    • Togge Answer

    use let if a variable might be reassigned.

    use const if a variable will not be assigned.

    var should not be used other than in older versions of JS.

  • What are the rules for naming variables?

    • Togge Answer

    Variable names must begin with a letter, an underscore (_) or a dollar sign ($).

    Variable names cannot contain spaces.

    Variable names are case-sensitive.

    Variable names can be any length.

    Variable names cannot be a reserved JS Keyword.

  • What are operators, operands, and operations?

    • Togge Answer

    An operator performs the operation on single or multiple operands.

    An operand is the data to be manipulated or operated on.

    An operation is the action carried out.

  • What is concatenation and what happens when you add numbers and strings together?

    • Togge Answer

    concatenation is the process of appending one string to another.

    When a number is added to a string the output will be a string.

  • What is the difference between == and ===?

    • Togge Answer

    When using == in JavaScript we are testing for loose equality. Double equals also performs type coercion.

    for === both the type and the value we are comparing have to be the same.

  • What are operator precedence values?

    • Togge Answer

    Operator precedence determines the order in which operators are evaluated. Operators with higher precedence are evaluated first.

  • What are the increment/decrement operators?

    • Togge Answer

    Increment ++

    Decrement --

  • What is the difference between prefixing and post-fixing them?

    • Togge Answer

    ++ Pre increment will increment the value before assigning it to the variable.

    ++ Post-increment will return the variable value first, then increment value by 1.

    -- Pre decrement decrements the value before assigning it to the variable.

    -- Post decrement will return the variable value first, then decrement value by 1.

  • What are assignment operators?

    • Togge Answer

    An assignment operator assigns a value to its left operand based on the value of its right operand. Example: (X = 3)

  • What is the “Unary +” Operator?

    • Togge Answer

    The Unary plus operator (+) Tries to convert the operand into a number.
